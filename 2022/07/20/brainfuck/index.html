HTB: Brainfuck

k1ckstand
2022-07-20
HTB
cve, john, linux, rsa, ssh2john, vigenere-cipher, wordpress

Brainfuck is an insane box where a lot of information gathering and exploits that need to be chained together. To start with the main site is running a vulnerable word press plugin that can give us admin access. From there we get an SMTP username and password stored on the site. Next we connect to the email server where more credentials are found. Looking at the certificate from the original site we discover two new subdomains, one of which takes to a “secret form”. Using the credentials found in the email we can login in and access the form, but one page is encrypted. The encryption turns out to be a vigenere cypher that we can decrypt using known text found in the signature of one of the users. The decrypted message gives us access to a private ssh key that will need to be cracked to get on the system. Getting the root flag involves solving some RSA. Let’s get started.

Getting User

Start out by adding the box to our host file.

Recon

Namp found a web server running exclusively on port 443 (https) @ brainfuck.htb. The server also looks to be doing email with smtp, imap, and pop3.

Taking a look at https://brainfuck.htb we can see it’s a wordpress site.

One thing of note here is that the dev update mentions “SMTP Integration is ready. Please check and send feedback to orestis@brainfuck.htb“.

Looking at the certificate on the page we find 2 new subdomains.

Add the new domains to the /etc/hosts file and let’s check them out.

https://www.brianfuck.htb resolves to the same page as http://brainfuck.htb but https://sup3rs3cr3t.brainfuck.htb takes us… to the super secret form.

Wordpress PrivEsc

As noted above the sit is running word press. Let’s run wpscan to get a bit more information.

There is a lot of output from wpscan and would guess more than one finding would work. We’re going to take a look at WP Support Plus Responsive Ticket System first. Another point of interest from wpscan is that 2 users were found: admin and administrator

Taking a closer look at php/webapps/41006.txt.

As is somewhat described above this exploit works by setting a admin cookie. We will need to make a few edits to the POC to get this working. We can use the username admin found from the wpscan and the email orestis@brianfuck.htb noted on the site.

To run the exploit type python3 -m http.server in the same directory as exploit.html. Next go to localhost:8000 in a new tab

Use admin as the username and click the Login button. Now go back to the http://brainfuck.htb tab and refresh the page. It should note at the top that you are now logged as admin.

Open the wordpress dashboard. Looking under settings -> Easy WP SMTP we find a username and password

Connecting to the email server is successful and we get an email with some credentials.

The Secret Form

Logging into the secret form:

The Development link has nothing of relevance.

The ssh access form has nothing directly important here but taking note of orestis signature (Orestis - Hacking for fun and profit) will come in handy later.

The last link takes us to the “key” form. This is an encrypted form.

It turns out this is a Vigenere Cipher. This works by “adding” the plain text message to a key phrase.

For example:
PT = abc
KEY = def

chr(ord(‘a’) + ord(‘d’) - ord(‘a’)) = ‘d’
chr(ord(‘b’) + ord(‘e’) - ord(‘a’)) = ‘f’
chr(ord(‘c’) + ord(‘f’) - ord(‘a’)) = ‘h’

CT = dfh

Let’s write this up in python. Keep in mind that to reverse the cipher we will need to subtract instead of add. We also need to account for wrap around so if the subtraction takes us past ord(‘a’) we need to go to ‘z’

The output from the script is brainfuckmybrainfuckmybrainfu
The key turns out to be fuckmybrain but the script started out with brain due to the previous offset of the message.

Now that we have the key we can decrypt the form messages using https://www.dcode.fr/vigenere-cipher.

The decoded message gives us a link to a private ssh key

Using ssh2john we can convert this key into a format that john can crack.

Now we can crack the key.

password is 3poulakia!

Now we can ssh onto the box.
