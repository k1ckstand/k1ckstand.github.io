Don’t Panic

The name of the above file is 5641444552.txt. Translated from hex: VADER

Every sentence in the file has a positive or negative sentiment. This can be determined programmatically with https://github.com/cjhutto/vaderSentiment. There were a few lines that were somewhat undecided so I made 2 separate list and hashed out the results rather than manually guess.

default positive: flig{7h<nks_v0r_4ll_th;_f35H}
default negative: flic{7h<nk3_f0_4ll_th;_f15H}
Flag: flag{7h<nks_f0r_4ll_th;_f15H}

Object Character Frustration

It’s a bit hard to tell but the yellow text is just a bunch of hex values. To solve each value needs to be read individually and written to a file as bytes. After having some initial difficulty with pytesseract I went with a manual approach of hashing every unique digit and creating a lookup table.

Characters of Shakespeare’s Plays

This challenge provides a text file of Shakespeare.

After some quick googling I found the what looks to be the original source of the provided file at https://www.gutenberg.org/cache/epub/5085/pg5085.txt. A quick look at the difference starts to spell out the flag.

Flag: flag(It's not the notes you play, it's the notes you don't play)
